# 🃏 Memory Match

カードを2枚めくって同じ絵柄のペアを見つけるメモリーゲーム。

## 遊び方
1. カードをクリックしてめくる
2. 2枚めくって同じ絵柄ならマッチ（そのまま残る）
3. 違う絵柄なら元に戻るので記憶して次へ
4. すべてのペアを見つけたらクリア！

## 難易度
| 難易度 | グリッド | カード枚数 |
|---|---|---|
| かんたん | 4×3 | 12枚（6ペア）|
| ふつう | 4×4 | 16枚（8ペア）|
| むずかしい | 6×4 | 24枚（12ペア）|

## 特徴
- 3段階の難易度
- クリアタイム・手数のカウント
- カードめくりの3Dフリップアニメーション
- マッチ時の緑ハイライト演出
- スコア表示（タイム・手数）

## 技術スタック
HTML / CSS（3D Transform） / Vanilla JavaScript
